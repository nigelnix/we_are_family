Fabulous Fonts!

The world of fonts...

Find the font that works best for you! But first, some rules...

  • Do not use more than 3-4 fonts.
  • Generally speaking, do not change the font mid-sentence.
  • Use sans-serif for online text and serif for print.
  • Use monospace for code.

Code

p {
color: green;
}
